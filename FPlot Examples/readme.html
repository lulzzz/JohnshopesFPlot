FPlot Examples

  • Fractals.fplot:
    A FPlot file conatining a library implementing algorithms for the Mandelbrot & Julia sets, and corresponding 2D functions that plots them.
  • Gauss data.txt:
    An ASCII File conatining example Measurement data. To Load the data, go to the Menu Edit->New Data... . The following Window will appear:


    In the topmost row of the grid, you must enter "n" in the x-column, "0.3" in the dx-column and "y[n]*0.3" in the dy-column. Then you must press the Load from file... button. The following window will show up:



    Next you click on Load file... . Here you must select the file "Gauss data.txt" and the gauss data will be loaded into FPlot.
  • Gaussfit.fplot:
    A FPlot file containing the above measurement data and a function with 3 gauss-curves that can be fitted to the data. To fit the function to the data go to the Menu Calc->Fit... . The following window will appear:


    Here you can click on Start to start fitting.
  • Integrate.fplot:
    A FPlot file containing a numerical integration example.
  • Sin.fplot:
    A FPlot file containing a sine curve
