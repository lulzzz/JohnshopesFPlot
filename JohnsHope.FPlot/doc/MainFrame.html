The Items Grid

Below the toolbar of the main window you see a grid showing all items that you have created thus far. If you click once on an item, you can select it. You can select multiple items by pressing the control and shift keys while clicking on an item. If you click on the Name cell of a selected item you can edit it's name. If you double click on the Name cell of an item you can edit this item in a separate window. If you doubleclick on the style cell of an item you can edit it's style.
