File menu:

New...

Creates a new FPlot session. After a confirmation dialog, all current data and plots will be discarded. Then a file dialog opens where you can choose a filename for the new FPlot session.

Open...

Opens an FPlot session file. A File Chooser dialog will appear, where you can select the file you want to open.

Save

Saves the FPlot session into a file with the name specified in the title-bar of the window.

Save as...

Saves the FPlot session into a file. A dialogbox will appear where you can choose a filename.

 Exit

Exits FPlot.


Edit menu:

New > Function...

Opens a dialogbox where you can edit a new function.

New > Data...

Opens a dialogbox where you can edit measurement data.

New > Library...

Opens a dialogbox where you can choose a file, and then a window where you can edit the library C# code.

Cut, Copy & Paste

Cuts, Copies and Pastes the selected items from the FPlot session.


Plot menu:

New 2D Plot...

Opens a new window showing a two dimensional plot of all checked items of the current FPlot session.

Apply Selection to Plot ...

Applies the current selection to the topmost plot, i.e. so that only the checked items are plotted in the plot.


Calc menu:

Fit...

Shows a dialogbox, where you can fit a function to measurement data. This menu item is only avalable, when there is a fittable function and some data.

Evaluate function...

Shows a dialogbox, where you can evaluate your functions at specific points.


Develop menu

Console Window...

Shows a console window where you can watch the output of any System.Diagnostics.Debug.Write or System.Console.Write calls, and where you can run C# statements.

Compiler Options

Shows a dialogbox, where you can set compiler options. If you use .NET libraries in your code, you must import them in the compiler options.


Help menu

Contents...

Shows this help.

About

Shows the about dialogbox. There you can register FPlot and can check the FPlot's version.
