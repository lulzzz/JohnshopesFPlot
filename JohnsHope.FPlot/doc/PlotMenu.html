File menu:

Save

Saves the plot with the filename that is shown in the title bar

Save As...

Opens a dialogbox where you can choose a filename and saves the plot with that filename

Save As Image

Opens a dialogbox where you can choose a filename and saves the plot into an image file of this name and file-type. Supported file types are bmp, gif, jpg, png, exif, tiff and emf (Metafile).

Print Preview...

Opens a dialogbox with a print preview.

Page setup...

Opens a dialogbox where you can configure the page setup used for printing.


Print...

Opens the dialogbox for printing the plot to a printer.

Options menu:

Plot Range...

Opens a dialogbox where you can set the plotting range.

Options...

Opens a dialogbox where you can set the appearance of the plot.


Help menu:

Contents...

Shows this help.

About

Shows the about dialogbox. There you can register FPlot and can check the FPlot's version.
