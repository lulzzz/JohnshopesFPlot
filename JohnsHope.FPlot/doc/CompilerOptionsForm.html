Edit compiler options

In this form you can enter compiler options. You can specify if the compiler should check for arithmetic overflow, include debug info, optimize code or allow unsafe constructs. You can also specify one of five warning levels to tell the compiler what warning messages it should produce. If you include debug info, when an exception occurs, FPlot will automatically show you where the exception occured.
You also need to tell the compiler which libraries to import. You do this by pressing the Imported libraries... button. You will be directed to the following dialogbox:

Here you can select the libraries you want to import. If you click on the glass icon, you can select a dll file. If you click on the cross icon you will delete the corresponding import.
